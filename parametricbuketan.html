Parametric Buketan

Description

Motive inspired by traditional Indonesian motive Buketan surrounded by rectangular parang. The flower is decorated with isen sawut and isen cecek. Formula to draw the flower is using parametric equation, stated by Hungarian math professor, Janos Karsai

Using formula developed by Janos Karsai, we can draw any n number of petal, with different kind of petal sharpness using just one general equation.

Math Concept

The formula to draw parametric flower, using polar equation can be described as:

\( r = A |sin(2n \theta)|^b - 0.25 A|sin(rn \theta)|^b + C \)

where n = number of petal

A = length of petal

b = control of petal waviness, 0.1 < b < 1

C = radius of flower inner circle
