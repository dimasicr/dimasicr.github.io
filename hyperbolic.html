Square Hyperbolic Kawung

Description

Motive inspired by traditional Indonesian motive kawung mapped conformally from poincare disk to square. Inspired by great paper "Analytical Methods for Squaring the Disc" written by Chamberlain Fong

Math Concept

We can use this formula to map from circle to square:

\( x = Re( \frac{1-i}{-K_e} F(cos ^{-1}(\frac{1+i}{\sqrt{2}} (u + vi)), \frac{1}{\sqrt{2}})) + 1 \)

\( y = Im( \frac{1-i}{-K_e} F(cos ^{-1}(\frac{1+i}{\sqrt{2}} (u + vi)), \frac{1}{\sqrt{2}})) - 1 \)

How to Draw

1. Given any polygon, we can draw any n-leaf polygon using simple algorithm

2. First, we determine the center of polygon.

3. Second, we draw bezier curve from polygon center, mid point of polygon verticex, and the vertices.
