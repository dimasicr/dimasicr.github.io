Geometry of Batik

Poincare Disk Kawung

Description

Motive inspired by traditional Indonesian motive Kawung tiled in Poincare Disk. is a model of 2-dimensional hyperbolic geometry in which all points are inside the unit disk, and straight lines are either circular arcs contained within the disk that are orthogonal to the unit circle or diameters of the unit circle.

This work is inspired by M.C Escher Circle Limit, but modified to use traditional Indonesia pattern.

Math Concept

We can determine whether {n,k} tessellation (Note: polygon n degree will meet k number of other polygon at each vertex ) will be tesselation in Euclidean plane, the hyperbolic plane, or the elliptic plane by using this formula:

For Euclidean tessellation, \( \frac{1}{n} + \frac{1}{k} = \frac{1}{2} \)

For Hyperbolic tessellation, \( \frac{1}{n} + \frac{1}{k} < \frac{1}{2} \)

For Elliptical tessellation, \( \frac{1}{n} + \frac{1}{k} > \frac{1}{2} \)
